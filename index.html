Introduction To Project

For this project, I will be showing how to send requests to mongoDB to retrieve questions, then send those questions to ChatGPT to receive an answer. MongoDB is a NoSQL database that uses documents similarly to JSON to store its data. I will be using mongoose to create a connection to MongoDB and be able to retrieve the data within its collections. Afterwards, this data will be sent to ChatGPT using axios to make HTTP requests from node.js. In the section below, I will show the average response times and accuracy of ChatGPT.

ChatGPT Efficiency

Research Approach: Here I will be describing the tools used for this assignment.

   1. MongoDB: A non-relaional (NoSQL) database management system that uses JSON-like documents to store and manage data.

   2. Node.js: Used for server-side scripting. It also provides NPM as a package manager.

   3. Express.js: A minimalist web framework for Node.js. It simplifies server creation, routing, and allows use of middleware. It is the backbone for many web applications and RESTful APIs.

   5. NPM (Node Package Manager): Package manager for Node.js, assists with discovery, installation, and management of javascript packages.

   6. Mongoose: Object Data Modelling library for MongoDB and Node.js. It manages relationships between data and provides schema validation.

   7. Axios: Promise-based HTTP Client for Node.js. It allows the user to make HTTP requests from node.js, such as fetching data from a database. It also performs error handling for the user.

   8. Xlsx: This was used to upload questions and answers from an Excel sheet onto MongoDB directly.

Key Findings:

Social Science: ChatGPT seems to struggle with answering social science questions accurately. This may be due to social science topics having more complex theories and interpretations and a lack of definitive answers.

History: ChatGPT seems to take longer on history questions, likely due to the longer responses that it generates, as history questions tend to require more information to answer. It is also more accurate in history questions as information on history is documented and easier to verify.

Computer Security: ChatGPT was fastest to respond to computer security questions, likely due to computer security having defined principles that are easier to explain in shorter terms. The information on computer security is also more recent and more easily accessable in comparison to history and social science.

Overall:

ChatGPT was approximately 60% accurate in its answers. Its response time is around 1 second per question. ChatGPT4 was interestingly slower than ChatGPT3.5, likely due to attempting to respond with longer answers.
